# Как опубликовать Гностику на GitHub

Чтобы ваш сайт стал доступен всему миру (и вам по ссылке), выполните следующие шаги:

1. **Создайте новый репозиторий** на сайте [github.com](https://github.com/new). Назовите его `gnostica`.
2. **Свяжите локальную папку с GitHub**:
   В терминале (я могу это сделать за вас, если вы дадите ссылку на репозиторий) нужно будет запустить:
   ```bash
   git remote add origin https://github.com/ВАШ_ЛОГИН/gnostica.git
   git branch -M main
   git push -u origin main
   ```
3. **Включите GitHub Pages**:
   В настройках репозитория на GitHub перейдите в раздел **Settings > Pages** и выберите ветку `main` в качестве источника.

После этого ваш сайт будет доступен по адресу: `https://ВАШ_ЛОГИН.github.io/gnostica/`
